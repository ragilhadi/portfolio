Project Overview

These project is to detect the plate with label name of "papan" using YoloV5, these project only detect 1 class called "papan" . after that take the bounding box and get the text inside the bounding box. to get the text inside bounding box these project use 2 method first using my own trained CNN model and used the python package name easyocr .

Object Detection Data Preparation

resize for all images and proceed with annotating objects on all images on the dataset using makesense.ai and getting the result of annotating objects with the necessary .txt format when practicing using Yolov5

What is YoloV5 ?

YOLOv5 🚀 is a family of compound-scaled object detection models trained on the COCO dataset, and includes simple functionality for Test Time Augmentation (TTA), model ensembling, hyperparameter evolution, and export to ONNX, CoreML and TFLite

Object Detection Using yoloV5

Yolov5 is used to detect objects with the class name "papan" that contains a room name. yolov5 can already detect "papan" objects very well

Image Charachter Segmentation

objects that have been detected by YoloV5 are then preprocessed to get the characters on the object using the opencv library.

Text Charachter Classification using Convolutional Neural Network

Convolutional Neural Network (CNN) is used for the classification of charachters found in extracted images. CNN conducted 10 class targets to ease the classification process.

Combine Yolov5 and CNN to detect and read text

CNN and Yolov5 combined to be able to detect objects and try to read the charachter contained in the object and display the results in images and sounds.

Block Diagram System

The system uses the camera as input to get the image then the image is processed using the Raspberry Pi and the result will be displayed on the Monitor and through the sound output

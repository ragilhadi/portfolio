Myocardial Infarction Detection System Using Arduino

"Arduino project to detect a person suffering from myocardial infraction by using pathological Q Waves and ST Segment Elevation as features"

Github Repository

What is Myocardial Infraction

Myocardial infraction occurs when one or more areas of the heart do not get enough oxygen supply due to blocked blood flow to the heart. Blocked blood flow to the heart is caused by plaque buildup in the arteries (atherosclerosis). When plaque ruptures or disintegrates it will form a blood clot, this blood clot is the cause of Myocardial Infraction.

Detect the Myocardial Infraction based on ECG Singnal

People with myocardial infraction can be detected by performing electrocardiography tests and reading the results of the signal. Because in myocardial infarction sufferers have some differences with normal signal results.

Using Pathological Q Waves & ST Segment Elevation as Features

Some of the differences that can be seen in ECG signals for myocardial infraction sufferers are Pathological Q Waves and ST Segment Elevation. This condition distinguishes from normal signals can be seen in the following image.

Classify Using Support Vector Machine

Both conditions are then used as a feature to be used to classify myocardial infraction sufferers and normal using support vector machine. The result of the accuracy of the classification is 82.3%.

Arduino System Report

Then the results of the classification are connected with the hardware. It uses arduino uno as a process and AD8232 sensor as ECG single-lead sensor. and the output uses a 16x2 LCD
